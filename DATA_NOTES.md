# Data Notes

User/company data is excluded from this GitHub-ready copy.

Excluded categories:

- Raw and processed plant time-series CSV files.
- Manual labels, generated drift reports, and exported HTML reports.
- Trained model artifacts such as `.pkl` files.
- Virtual environments and Python caches.
- Delivery folders that may contain private project context.

The repository keeps algorithmic code, notebooks, pipeline modules, configuration examples, and documentation. To reproduce experiments, create a local `data/` folder and use anonymized or public sample data.
